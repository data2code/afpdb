To test and debug, run

If you don't have python

	Follow https://docs.anaconda.com/free/miniconda/ to install miniconda

If you don't have pytest
	pip install pytest

If you need jupyter notebook

	pip install notebook

If you need pymol

	conda install conda-forge::pymol-open-source

To install/reinstall afpdb

	in the afpdb root folder
	pip install .

To run test
	in the afpdb root folder
	pytest

To run jupyter notebook
	in the afpdb root folder
	jupyter notebook
	open jupyter/afpdb.ipynb
